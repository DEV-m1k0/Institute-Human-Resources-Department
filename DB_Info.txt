База данных:

1. Таблица с должностями:
    1. Id
    2. Название должности
    3. Зарплата

2. Таблица с кафедрами:
    1. Id
    2. Название кафедры

3. Таблица всех сотрудников:
    1. Id
    2. Имя
    3. Фамилия
    4. Отчество
    5. Логин
    6. Пароль
    7. Возраст
    8. Статус отпуска
    9. Статус пенсионера 
    10. Статус предпенсионного возраста 
    11. Статус бездетного сотрудника 
    12. Статус многодетного сотрудника 
    13. Статус ветерана
    14. Дата рождения

4. Таблица с вакансиями:
    1. Id
    2. Id Должности (связь с таблицей 1)
    3. Id Кафедры (связь с таблицей 2)
    4. Дата открытия вакансии

5. Таблица для хранения сотрудников и их должности(так как у сотрудника может быть более 2ух должностей. Это нужно, чтобы мы могли считать зарплаты сотрудников исходя из того, где и на какой должности сотрудник работает)
    1. Id 
    2. id Сотрудника (Связь с таблицей 3)
    3. Id Должности (Связь с таблицей 1)
    4. Id Кафедры (связь с таблицей 2)
